# Goodies

Copy as Markdown

Dokploy has certain goodies that are external that can be used with Dokploy.

1. Ansible Dokploy: Ansible role to deploy Dokploy Ansible Role
2. Dokploy Oracle infrastructure: Deploy Dokploy on Oracle infrastructure Github
3. Dokploy Deploy Action 1: Automatic Dokploy deployments on Github Github
4. Dokploy Deploy Action 2: Automatic Dokploy deployments on Github Github
5. Dokploy Deploy Action 3: Automatic Dokploy deployments on Github (Support both`application` and`compose` deployment) Github
6. Dokploy JS Sdk: Dokploy JS SDK Github
7. Templates Collection : Docker compose collection for Dokploy Github
8. Dokploy Port Updater: Dokploy Port Updater Github
9. Dokli TUI: Dokli TUI Github
10. nix-dokploy: A NixOS module that runs Dokploy using declarative systemd units Github
